# Incheon-Public-Data-Analysis

인하대학교 데이터사이언스학과와 미추홀구청 간 관학협력 기반 공공데이터 분석 프로젝트입니다.

## Overview

행정 데이터 기반 분석을 통해 정책 의사결정 지원 및 행정 서비스 개선을 목표로 합니다.

- 프로젝트 시작일: 2026.05.11
- 참여 기관:
  - 인하대학교 데이터사이언스학과
  - 미추홀구청

담당 과제:
- 구 소셜미디어 성과 분석

## Objectives

- 공공데이터 기반 행정 의사결정 지원
- 민원 및 행정 서비스 효율화
- 지역 현안 데이터 분석
- 정책 수립을 위한 인사이트 도출

## Expected Outcomes

- 데이터 기반 정책 제안
- 행정 업무 개선 방향 도출
- 시각화 기반 결과 보고서 작성
- 공공데이터 활용 사례 구축
